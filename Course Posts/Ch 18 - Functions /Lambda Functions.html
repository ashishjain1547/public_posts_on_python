• Lambda Function:
  • It is a small anonymous function.
  • It can take any number of arguments but have only one expression inside it
  • It should use when an anonymous function required for a short period of time
  • Syntax

  • lambda arguments : expression

  • The expression is executed and the result is returned: For Ex –
  • 1) Multiply argument a with argument b and return the result:
    • x = lambda a, b : a * b 
    
print(x(5, 6))
  • output will be : 30
  • 2) Summarize argument a, b, and c and return the result:
    • x = lambda a, b, c : a + b + c 
    
print(x(5, 6, 2))
  • output will be : 13

  • Why Use Lambda Functions?
  • Its power reflects when they are used as anonymous function inside another function. For Ex –
  • 1) Defining a function which take only one argument and that will be multiplied with an unknown number
