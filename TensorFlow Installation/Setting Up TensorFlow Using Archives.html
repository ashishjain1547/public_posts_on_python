Step 3:
Create the .bat file named "TensorFlow_Installation_Script.bat" with following contents:

pip --no-cache-dir install termcolor-1.1.0.tar.gz
pip --no-cache-dir install six-1.12.0-py2.py3-none-any.whl
pip --no-cache-dir install protobuf-3.6.1-cp35-cp35m-win_amd64.whl
pip --no-cache-dir install absl-py-0.6.1.tar.gz
pip --no-cache-dir install numpy-1.15.4-cp35-none-win_amd64.whl
pip --no-cache-dir install gast-0.2.0.tar.gz
pip --no-cache-dir install h5py-2.9.0-cp35-cp35m-win_amd64.whl
pip --no-cache-dir install Keras_Applications-1.0.6-py2.py3-none-any.whl
pip --no-cache-dir install Keras_Preprocessing-1.0.5-py2.py3-none-any.whl
pip --no-cache-dir install Werkzeug-0.14.1-py2.py3-none-any.whl
pip --no-cache-dir install Markdown-3.0.1-py2.py3-none-any.whl
pip --no-cache-dir install grpcio-1.17.1-cp35-cp35m-win_amd64.whl
pip --no-cache-dir install tensorboard-1.12.1-py3-none-any.whl
pip --no-cache-dir install astor-0.7.1-py2.py3-none-any.whl
pip --no-cache-dir install tensorflow-1.12.0-cp35-cp35m-win_amd64.whl

Step 4:
Installation process is:
    1. Keep the above mentioned archives in a directory. Suppose it is "C:\archives", then do "cd C:\archives"
    2. Copy-paste the .bat file here in this directory.
    3. Launch “TensorFlow_Installation_Script.bat” from here in command prompt. Order in which to install these packages is important.
