4. Create a "Path" from Strings:

"os.path.join" joins strings with the character representing 'path separater' for the OS.

>>> os.path.join("DIR1", "DIR2")
'DIR1\\DIR2' 

>>> os.path.join("DIR1", "DIR2", "DIR3")
'DIR1\\DIR2\\DIR3' 

>>> os.path.sep
'\\'
